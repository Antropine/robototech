## 💡 О проекте

**RoboTech** — это образовательная платформа для детей, направленная на развитие технических навыков через:

- робототехнику  
- программирование  
- 3D-моделирование  

Проект выполнен в стиле **pixel / neon aesthetic** с кастомными UI-элементами, создающими уникальную цифровую атмосферу обучения.
